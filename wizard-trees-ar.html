You're currently in local demo mode. To share data across the team:
  1. Open apps-script.gs (included with this file) and follow the instructions at the top to deploy it as a Web App in Google Apps Script.
  2. Copy the Web App URL it gives you.
  3. Open this HTML file in a text editor and paste the URL into the SCRIPT_URL variable near the top of the <script> section.
  4. Save and re-open — contacts, reps, and sent log will now sync for everyone.
